## Difference Array Pattern
Mark changes at event boundaries
Sweep sorted positions to get running total
Use for: intervals, scheduling, capacity

changes[start] += value
changes[end]   -= value
sweep sorted(changes) → running total

Applications:
→ Car pooling (capacity over time)
→ Meeting room booking
→ Spark streaming window counts!
